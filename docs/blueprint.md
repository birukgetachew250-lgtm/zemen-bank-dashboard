# **App Name**: Zemen Admin Center

## Core Features:

- Dashboard Overview: Display key metrics, charts, and recent activities related to customer registrations and transactions.
- Customer Management: Enable admin staff to review, approve, reject, or update customer profiles, accounts, and permissions, with full audit trail.
- Corporate Management: Dedicated portal to manage corporate accounts, view transaction summaries and detailed records.
- Transaction Reporting: Generate comprehensive reports on transaction volumes, successful and failed transactions based on a date range.
- System Reporting: Produce in-depth system reports on registered, active, inactive, failed and dormant customers.
- User and Role Management: Handle roles, permissions, branches, departments, and other settings.
- Anomaly Detection: Leverage AI to analyze transaction patterns and flag any unusual or suspicious activities.

## Style Guidelines:

- Primary color: Zemen Red (#D02149) to maintain brand identity.
- Background color: Light gray (#F4F4F4) for a clean, modern feel.
- Accent color: Dark charcoal (#4A55A2) for UI elements that need emphasis. 
- Body and headline font: 'Poppins' sans-serif, as requested.
- Note: currently only Google Fonts are supported.
- Consistent icon set for navigation and data representation.
- Collapsible sidebar with intuitive navigation for key admin functions.
- Subtle background watermark with the bull image, adding depth without distraction.
- Smooth transitions and loading animations to improve user experience.
